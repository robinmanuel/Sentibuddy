# SentiBuddy

### Backend

1. Go to backend directory
   `cd backend`

2. Install dependencies
   `pip install -r requirements.txt`

3. Run Flask Server
   `python app.py`

### Frontend

1. Go to frontend directory
   `cd frontend`

2. Install node modules
   `npm install`

3. Run development server
   `npm run dev`

4. Open website at http://localhost:5173/ (Default)
